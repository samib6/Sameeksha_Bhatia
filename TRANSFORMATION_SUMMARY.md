## ğŸ¨ Customization Tips

### Want to Change Colors?
The primary green (#207561) is used throughout:
- Search: `#207561`
- Replace with your brand color
- Also update `#0d3f31` (dark variant)

### Want to Adjust Animations?
Find these in CSS:
- `transition: all 0.3s ease` (duration)
- `transform: translateY(-8px)` (movement)
- `@keyframes` definitions (animation styles)

### Want to Add More Projects?
Copy this template in the portfolio section:
```html
<div class="col-lg-4 col-md-6 mb-4">
  <div class="portfolio-card">
    <div class="portfolio-image">
      <img src="image.jpg" alt="Project">
      <div class="portfolio-overlay">
        <a href="image.jpg" class="portfolio-link fancybox">
          <span class="icon-search2"></span>
        </a>
      </div>
    </div>
    <div class="portfolio-content">
      <div class="portfolio-category">Category</div>
      <h3>Project Title</h3>
      <p>Description</p>
      <div class="tech-stack">
        <span class="tech-badge">Tech1</span>
        <span class="tech-badge">Tech2</span>
      </div>
    </div>
  </div>
</div>
```

## ğŸ“ Technical Support

### Common Issues

**SVG not showing?**
- Check browser console for errors
- Ensure `xmlns` attribute is present
- Test in different browsers

**Animations stuttering?**
- Check browser performance
- Reduce animation count
- Use Chrome DevTools to profile

**Form not working?**
- Set up Formspree account
- Get form ID
- Update form action URL

**Mobile looks weird?**
- Clear browser cache
- Check viewport meta tag
- Test in different device sizes
